## Introduction

Machine Learning is a field of computer science aimed at enabling computers to learn from and make predictions based on data. This repository aims to provide resources and hands-on projects for anyone interested in exploring the concepts and applications of machine learning.

## Courses

### 1. Intro to Machine Learning
* Delivery Platform: Coursera
* Instructor: Andrew Ng
* Content: Covers basic ML concepts, algorithms, and applications.

### 2. Applied Data Science with Python
* Delivery Platform: Coursera
* Instructor: Multiple Instructors
* Content: Focuses on data visualization, machine learning, and data analysis using Python.

### 3. Deep Learning Specialization
* Delivery Platform: Coursera
* Instructor: Andrew Ng
* Content: Deep learning concepts, neural networks, and frameworks.

## Projects

### 1. Titanic Survival Prediction
* Description: Predict survival on the Titanic using machine learning classifiers.
* Tools: Python, Pandas, Scikit-learn

### 2. Handwritten Digit Recognition
* Description: Build a model to recognize handwritten digits using neural networks.
* Tools: Python, TensorFlow, Keras

### 3. Image Classification with CNN
* Description: Implementing Convolutional Neural Networks for image classification.
* Tools: Python, TensorFlow, Keras
